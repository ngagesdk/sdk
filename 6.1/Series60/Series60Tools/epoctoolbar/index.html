Epoc Toolbar

The visual studio toolbar adds a toolbar to visual studio. The toolbar has buttons to do the following:

  1. Create a new EPOC class
  2. Update the visual studio workspace from the MMP
  3. Build for target
  4. Memory Tracker
  5. Generate a pkg file for building a sis

Installation

To install this utility do the following:
  1. Copy the EpocTools.dll file in this directory to:
    >your visual studio<\Common\MSDev98\Addins
  2. Start up Microsoft Visual Studio
  3. Select Tools > Customize from the menu bar
  4. Click the 'Add-ins and Macro Files' tab
  5. Check the box next to 'EPOC Toolbar Add-in'
  6. Click the Close button
The EPOC toolbar should the appear and can be dragged onto the main toolbar.
