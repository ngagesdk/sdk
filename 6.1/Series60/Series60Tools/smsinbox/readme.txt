Running SMS Inbox:
==================

1. Copy SMS Inbox epoc32 -directory on top of the SDK epoc32 -directory 

2. Start emulator and the SMS inbox application in the emulator

3. Leave the SMS inbox application running on the back while testing the SMS sending/receiving.
    (press applications key)

4. When SMS is sent the sms is copied to smsout folder.

5. For receiving SMS the message is copied to smsin folder.
    The SMS inbox application automatically recoqnize the messages from smsin folder.

6. Normal SMS viewer and editor can be used for editing sms messages.
