Menu summary

The menu bar commands are summarised in table below.

Menu Bar

Menu Item

Description

File

New Project…

Creates a new project. For this and other File menu commands, see How to create, load and save projects.

Open Project…

Opens a project from disk

Close project

Closes the currently selected file

Save Project

Saves the project file

Save Project As

Save the project file with a new name

Exit

Exits the application

Minimize

Minimises the application's main window

Tools

Build SIS File

Builds the .sis file

Import PKG File…

Imports the information from the selected .pkg file

Import AIFB File…

Imports the information relevant to the project from the selected .aifb file

Default Configuration

Brings up a form for editing the global options of the application
