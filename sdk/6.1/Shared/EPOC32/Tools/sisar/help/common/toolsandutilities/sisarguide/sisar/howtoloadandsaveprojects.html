How to create, load and save projects

For each application for which you want to use Sisar, create a new Sisar project file (extension .sisar) using the New project... command on the File menu.

You can save the information you enter using Save and Save Project As..., and load it again later using Open Project....

Note that for Open Project... batch mode supports multiple project files, but GUI mode does not. The GUI file selector does not allow the user to select more than one file.
